S2F Analytics Dashboard
High-fidelity analytics web component for chat and booking insights
Web Component TypeScript AG Grid Analytics

Quick Start

Add the S2F Analytics Dashboard to any website with just 2 lines of code. No configuration needed - all CSS and dependencies are automatically loaded!

HTML 
<!-- Add this script tag -->
<script type="module" src="https://your-netlify-site.netlify.app/s2f-analytics.js"></script>

<!-- Add the component -->
<s2f-analytics-dashboard s2fid="pd-northern-nj"></s2f-analytics-dashboard>

âœ¨ That's it! The component automatically loads all required CSS and JavaScript dependencies (Remix Icons, AG Grid, custom styles).

Important: Non-Encapsulated Component
This web component is not encapsulated - it does not use Shadow DOM. This means:
  • Styles are automatically injected into the page when the script loads
  • Parent page styles can affect the component's appearance
  • You can customize styling by adding CSS rules after the component loads
  • No manual CSS imports needed - everything is bundled!

Here's a complete HTML page example showing how to integrate the S2F Analytics Dashboard.

</html>

Component Attributes

Configure the dashboard component using HTML attributes.

  • s2fid string required
    Market identifier for filtering data. This determines which market's data to display in the dashboard.

    Example: s2fid="pd-northern-nj"
Pro Tip
You can dynamically change the market by updating the s2fid attribute with JavaScript: 
document.querySelector('s2f-analytics-dashboard').setAttribute('s2fid', 'new-market-id');

Features

The S2F Analytics Dashboard includes powerful features for analyzing your chat and booking data.

  • Real-time KPIs
    Track total chats, conversion rates, average duration, and revenue opportunities in real-time.
  • Channel Performance
    Analyze performance across SMS, Voice, and Web chat channels with conversion tracking.
  • Attribution Analytics
    Track marketing effectiveness by source (Google, Bing, GMB, Facebook, etc.).
  • Advanced Filtering
    Filter by booking status, customer type, date ranges, and attribution sources.
  • Transcript Viewer
    View complete conversation transcripts with markdown formatting support.
  • Customer Profiles
    Access detailed customer information including contact details and booking history.

Styling & Customization

The dashboard background is transparent by default to blend seamlessly into existing applications. Wrap the component in any container element to control layout and styling.

Wrapper Pattern
The web component has a transparent background, allowing it to integrate into any page design. Use a wrapper element (like .analytics-container) to control the background, spacing, and layout: 
<div class="analytics-container">
  <s2f-analytics-dashboard s2fid="your-market-id"></s2f-analytics-dashboard>
</div>

<style>
  .analytics-container {
    background: #0a0f1a; /* Your background color */
    padding: 20px;       /* Optional spacing */
  }
</style>
The class name can be anything you choose - use whatever fits your application's structure.

The dashboard uses CSS variables for easy customization. Override these variables in your page's CSS to match your brand.

CSS

Background Control
The dashboard container has a transparent background by default. You can add a background to the parent element or wrap it in a styled container: 
<div style="background: linear-gradient(135deg, #0f172a 0%, #1e4244 100%); padding: 20px;">
  <s2f-analytics-dashboard s2fid="your-id"></s2f-analytics-dashboard>
</div>

CDN Alternatives

If you prefer to self-host or use a different CDN, you can build and deploy the component yourself.

Bash 
# Clone the repository
git clone https://github.com/your-repo/s2f-analytics.git

# Install dependencies
npm install

# Build for production
npm run build

# The built files will be in the 'dist' folder
# Upload dist/s2f-analytics.js to your CDN or server
