S2F Analytics Dashboard
High-fidelity analytics web component for chat and booking insights
Web Component TypeScript AG Grid Analytics

Quick Start

Get started in under 2 minutes. Add the S2F Analytics Dashboard to any website with just a script tag and HTML element.

  1. Add the script tag to your HTML
    HTML 
    <script type="module" src="https://your-netlify-site.netlify.app/s2f-analytics.js"></script>
  2. Add the component to your page
    HTML 
    <s2f-analytics-dashboard s2fid="pd-northern-nj"></s2f-analytics-dashboard>
  3. Done!

    The dashboard will load and display your analytics data.

Important: Non-Encapsulated Component
This web component is not encapsulated - it does not use Shadow DOM. This means:
  • Parent page styles can affect the component's appearance
  • The component's styles are loaded via external CSS files
  • You can customize styling by modifying the parent page's CSS

For style changes or application updates, please contact conner@a2opinion.com

Complete Example

Here's a complete HTML page example showing how to integrate the S2F Analytics Dashboard.

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Analytics Dashboard</title>

  <!-- S2F Analytics Dashboard -->
  <script type="module" src="https://your-netlify-site.netlify.app/s2f-analytics.js"></script>

  <style>
    body {
      margin: 0;
      padding: 20px;
      font-family: system-ui, sans-serif;
      background: #0f172a;
    }
  </style>
</head>
<body>
  <!-- Analytics Dashboard Component -->
  <s2f-analytics-dashboard s2fid="pd-northern-nj"></s2f-analytics-dashboard>
</body>
</html>

Component Attributes

Configure the dashboard component using HTML attributes.

  • s2fid string required
    Market identifier for filtering data. This determines which market's data to display in the dashboard.

    Example: s2fid="pd-northern-nj"
Pro Tip
You can dynamically change the market by updating the s2fid attribute with JavaScript: 
document.querySelector('s2f-analytics-dashboard').setAttribute('s2fid', 'new-market-id');

Features

The S2F Analytics Dashboard includes powerful features for analyzing your chat and booking data.

  • Real-time KPIs
    Track total chats, conversion rates, average duration, and revenue opportunities in real-time.
  • Channel Performance
    Analyze performance across SMS, Voice, and Web chat channels with conversion tracking.
  • Attribution Analytics
    Track marketing effectiveness by source (Google, Bing, GMB, Facebook, etc.).
  • Advanced Filtering
    Filter by booking status, customer type, date ranges, and attribution sources.
  • Transcript Viewer
    View complete conversation transcripts with markdown formatting support.
  • Customer Profiles
    Access detailed customer information including contact details and booking history.

CDN Alternatives

If you prefer to self-host or use a different CDN, you can build and deploy the component yourself.

Bash 
# Clone the repository
git clone https://github.com/your-repo/s2f-analytics.git

# Install dependencies
npm install

# Build for production
npm run build

# The built files will be in the 'dist' folder
# Upload dist/s2f-analytics.js to your CDN or server
